# Tic_Tac_Toe

A simple CLI TicTacToe Game 
Two Player Game
 
---


## Requirements

1. make installed

2. Compiler installed : GNU gcc or clang


---
 
## Run the game

1. Clone the repo
2. Open the terminal
3. Navigate to the cloned repository
4. * If you have gcc installed run this command :
          ```make g++ run```
   * If you have clang installed run this command :
          ```make clang run```
